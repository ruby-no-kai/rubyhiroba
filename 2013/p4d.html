P4D (デザイナー向けプログラム部)

P4D (デザイナー向けプログラム部) はデザイナーとプログラマーで集まって Web や アプリをつくったりデザインやプログラミングについて相談したりするコミュニティです。

RubyHiroba 2013 では Web デザインを学びたい Rubyist のためのワークショップを実施します。

ワークショップに参加される方は、可能であれば事前に以下のリポジトリの clone し、bundle install まで済ませてください！
(やり方がわからない場合は、当日、開始前に聞いていただければ出来る範囲でお手伝いします。)

  • https://github.com/prog4designer/zurui-hands-on

内容

  • ズルいデザイン実践講座 by @ken_c_lo
  • Web サイト、Web サービスの相談事コーナー

対象

Web デザインのスキルを身につけたい Rubyist

持ち物

実際に手を動かしてデザインを学ぶ課題を準備しますので、無線 LAN に接続が出来る PC / Mac をお持ちください。

ご自身で作られた Web サイトやサービスのデザイン相談をご希望の方はサイト / サービスの簡単な説明と具体的な相談内容を準備していただけると幸いです。

参加方法

直接会場にお越し下さい。

参加希望者多数の場合は、座席の都合上先着順となりますのでご了承ください。

なお、事前に以下のリポジトリを clone し、bundle install まで済ませていただくと、スムーズにご参加いただけると思います。

ご協力をお願いいたします。
